5  Accelerated Failure Time (AFT) Models

CautionAims for this chapter

In this chapter, we will study a class of regression models, Accelerate Failure Time (AFT) models, for relating the survival time of an individual to explanatory variables for that individual. In particular we will

  1. study the defining characteristics of an AFT model: how explanatory variables ‘accelerate’ an individual along the time axis;

  2. understand how the parameters of an exponential AFT model are estimated using maximum likelihood;

  3. use AFT models to predict survival times;

  4. fit AFT models using R, obtain parameter estimates, and conduct hypothesis tests to see which explanatory variables are significant.

5.1 Motivation

So far we have just looked at problems when all the failure times of all the individuals are assumed IID or at most there are a fixed number of distinct groups with a common failure time distribution for each group. However, each individual might have their own failure time distribution dependent upon a number of characteristics about either them or their treatment. For a more sensitive analysis, we might want to include in our failure-time model the possible effect of these explanatory variables. Examples of such variables might be:

  • Treatment given (typically binary or categorical if multiple treatments are considered)

  • Age of individual (continuous)

  • Sex of individual (categorical)

  • Measured variables describing prior medical history

For each individual, with explanatory variables \(\mathbf{x}\) we may wish to allow their survival time to depend upon these variables. When we do so, we typically define \(\mathbf{x} = \mathbf{0}\) to correspond to a baseline and then measure \(\mathbf{x}\) relative to this baseline.

5.2 A Simple Two-Sample Example

Consider initially the example where we have \(n\) individuals belonging to one of two groups dependent upon whether they are controls (Group 0) or receive a treatment (Group 1). In this case the binary variable \[ \mathbf{x_i} = \left\{ \begin{array}{rl} 0 & \textrm{if person $i$ is in the control group} \\ 1 & \textrm{if person $i$ is in the treatment group} \end{array} \right. \] denotes which group the \(i^{th}\) individual belongs to. Suppose the failure times of the individuals in the control and treatment groups are labelled as \(T_0\) and \(T_1\) respectively. If \(S_0(t)\) and \(S_1(t)\) are the survivor functions then we could choose to model them as \[\begin{align} \text{Group 0:}& \quad S_0(t) = P(T_0 > t) \\ \text{Group 1:}& \quad S_1(t) = P(T_1 > t) = P(T_0 > \psi t) = S_0(\psi t) \end{align}\] so that, in terms of random variables \(T_1 = T_0/\psi\). Intuitively this means that lifetime is accelerated/decelerated in group 1 compared with group 0. Any individual with survival time \(t\) in group 0 would have survival time \(t/\psi\) in group 1. If \(\psi > 1\) then individuals will fail more quickly in group 1 compared with group 0 while if \(\psi < 1\) then they will fail more slowly.

Note

We refer to \(S_0\) as the “baseline” survivor function, and the distribution of \(T_0\) as the baseline survival distribution. AFT models involve specifying how explanatory variables change this baseline distribution.

5.3 A General AFT Model Framework for including Explanatory Variables

Suppose instead that we have a vector of explanatory variables (\(\mathbf{x}\)) for each individual. We can extend this idea by modelling the survivor function of an individual with variables \(\mathbf{x}\) as \[\begin{equation} S(t;\mathbf{x}) = S_0(t \psi(\mathbf{x})) \end{equation}\] where \(S_0(t)\) corresponds to the survivor function of the chosen baseline \(\mathbf{x} = \mathbf{0}\) for any positive function \(\psi(\mathbf{x})\) (with \(\psi(\mathbf{0}) = 1\) to satisfy \(S(t;\mathbf{x}=\mathbf{0}) = S_0(t)\)). With this parameterisation, we have the density and hazard functions (as functions of \(t\) and \(\mathbf{x}\)) given by \[\begin{align} f(t;\mathbf{x}) &= f_0(t \psi(\mathbf{x})) \psi(\mathbf{x}) \\ h(t;\mathbf{x}) &= h_0(t \psi(\mathbf{x})) \psi(\mathbf{x}) \end{align}\]

In terms of random variables this is equivalent to defining \[ T = T_0 / \psi(\mathbf{x}), \] where \(T_0\) has survivor function \(S_0(\cdot)\). Since we require \(\psi(\mathbf{x}) \geq 0\) and \(\psi(\mathbf{0}) = 1\) a natural choice is \[ \psi(\mathbf{x}) = e^{-\boldsymbol{\beta}'\mathbf{x}}. \] For this choice of \(\psi(\mathbf{x})\), we can write \[ \log T = \boldsymbol{\beta}'\mathbf{x} + \log T_0. \]

Note

  • We can think of this of two equivalent ways to specify an AFT model: \[ S(t;\mathbf{x}) = S_0(t \exp(-\boldsymbol{\beta}'\mathbf{x})) \quad\Leftrightarrow\quad \log T = \boldsymbol{\beta}'\mathbf{x} + \log T_0, \] where \(T\) and \(T_0\) are two random survival times with survivor functions \(S\) and \(S_0\) respectively. Note that \(T_0\) is the survival time of an individual with covariates \(\mathbf{x}=\mathbf{0}\), and \(T\) is the survival time of an individual with some other value for the covariate vector \(\mathbf{x}\).

  • If we denote \(\log T_0\) by \(\varepsilon\), noting that \(\varepsilon\) is a random variable, and we write \[ \log T = \boldsymbol{\beta}'\mathbf{x} + \varepsilon, \] then we can see that we have specified a log-linear regression model that relates a log survival time \(\log T\) to some covariates \(\mathbf{x}\). If you have studied generalised linear models (GLMs), then note that this is not necessarily a GLM, as we may choose distributions for \(\varepsilon\) that are not in the exponential family, and there may also be a need to incorporate censored observations.

5.4 Choice of distribution in AFT models

To specify an AFT model, we need to make a choice of distribution: the survivor function \(S_0\), or equivalently, the distribution of \(\log T_0\). In theory, the only constraint is that we must have \(T_0 >0\), otherwise, any distribution can be used. However, when fitting an AFT model in R (using the survival::survreg() function), there are some restrictions, and the recommended choices are weibull (or exponential as a special case of the weibull), log normal or log logistic (i.e. \(\log T_0\) has a normal/logistic distribution.)

5.5 The exponential AFT model

The simplest AFT model is to assume an exponential distribution for the baseline distribution of \(T_0\). We then assume that each patient’s survival time has an exponential distribution, with a rate parameter that depends on the patient’s explanatory variables.

Suppose that we have \(n\) individuals where for each we have observed

\[\begin{align} t_i &= \min(\textrm{Failure time}, \textrm{Time of right censoring}) \\ \delta_i &= \left\{ \begin{array}{rl} 1 & \textrm{if $i$ is observed to fail} \\ 0 & \textrm{if $i$ is censored} \end{array} \right. \\ \mathbf{x_i} &= \textrm{$p$-dimensional vector of explanatory variables} \end{align}\] For an exponential AFT model, following the specification set out in Section 5.3, with the choice \[ \psi(\mathbf{x}) = e^{-\boldsymbol{\beta}'\mathbf{x}}, \] we have \[\begin{align} S_0(t)= e^{-\lambda t} & \quad \Rightarrow \quad S(t;\mathbf{x}) = e^{-\lambda t e^{-\boldsymbol{\beta}'\mathbf{x}}} \\ h_0(t)=\lambda & \quad \Rightarrow \quad h(t;\mathbf{x}) = \lambda e^{-\boldsymbol{\beta}' \mathbf{x}} \\ f_0(t)=\lambda e^{-\lambda t} & \quad \Rightarrow \quad f(t;\mathbf{x})=\lambda e^{-\boldsymbol{\beta}'\mathbf{x}}e^{-\lambda te^{-\boldsymbol{\beta}' \mathbf{x}}} \end{align}\] Note that this is equivalent to assuming that with explanatory variables \(\mathbf{x}\) the failure time \[T \sim Exp(\lambda e^{-\boldsymbol{\beta}' \mathbf{x}}).\]

5.5.1 Parameter estimation

We estimate the \(p+1\) parameters \(\lambda, \beta_1, \beta_2, \ldots, \beta_p\) by maximum likelihood. We write the likelihood as a product of terms representing the observed failures and a product of terms representing the censored observations. \[\begin{align} L(\lambda,\boldsymbol{\beta}) &= \prod_{i=1}^n f(t_i;\mathbf{x_i})^{\delta_i} S(t_i;\mathbf{x_i})^{1-\delta_i} \\ &= \prod_{i=1}^n \left[ \lambda e^{-\boldsymbol{\beta}' \mathbf{x_i}} e^{-\lambda t_i e^{-\boldsymbol{\beta}' \mathbf{x_i} }} \right]^{\delta_i} \left[ e^{-\lambda t_i e^{-\boldsymbol{\beta}' \mathbf{x_i}} } \right]^{1- \delta_i} \\ &= \prod_{i=1}^n \left[ \lambda e^{-\boldsymbol{\beta}' \mathbf{x_i}} \right]^{\delta_i} e^{ -\lambda t_i e^{-\boldsymbol{\beta}' \mathbf{x_i}}}. \end{align}\] Taking logs gives \[\begin{align} l(\lambda,\boldsymbol{\beta}) &= \log \lambda \sum_{i=1}^n \delta_i - \sum_{i=1}^n \delta_i \boldsymbol{\beta}' \mathbf{x_i} - \lambda \sum_{i=1}^n t_i e^{-\boldsymbol{\beta}' \mathbf{x_i}}. \end{align}\] Differentiating gives \[\begin{align} \frac{\partial l}{\partial \lambda} &= \frac{\Delta}{\lambda} - \sum t_i e^{-\boldsymbol{\beta}' \mathbf{x_i}} \quad \textrm{where } \Delta = \sum_{i=1}^n \delta_i \\ \frac{\partial l}{\partial \beta_j} &= -\sum_{i=1}^n \delta_i x_{ij} + \lambda \sum_{i=1}^n x_{ij} t_i e^{-\boldsymbol{\beta}' \mathbf{x_i}} \quad \textrm{for } j = 1, \ldots, p. \end{align}\]

The maximum likelihood estimates are the solutions of \[\begin{align} \frac{\Delta}{\hat{\lambda}} - \sum t_i e^{-\hat{\boldsymbol{\beta}}' \mathbf{x_i}} &=0,\\ -\sum_{i=1}^n \delta_i x_{ij} + \hat{\lambda} \sum_{i=1}^n x_{ij} t_i e^{-\hat{\boldsymbol{\beta}}' \mathbf{x_i}}&=0. \end{align}\]

We cannot solve these equations analytically, so numerical methods have to be used (R uses the Newton-Raphson method).

For estimates of variance and standard errors we can use the asymptotic properties of the mle and note that \[\begin{align} \frac{\partial^2 l}{\partial \lambda^2} &= -\frac{\Delta}{\lambda^2} \\ \frac{\partial^2 l}{\partial \lambda \partial \beta_j} &=\sum_{i=1}^n x_{ij} t_i e^{-\boldsymbol{\beta}' \mathbf{x_i}} &\quad \textrm{for } j = 1, \ldots, p \\ \frac{\partial^2 l}{\partial \beta_j \partial \beta_k} &=-\lambda \sum_{i=1}^n x_{ij} x_{ik} t_i e^{-\boldsymbol{\beta}' \mathbf{x_i}} &\quad \textrm{for } j = 1, \ldots, p; \quad k = 1, \ldots, p \end{align}\]

5.5.2 Model fitting in R

As an example, we will use the leuk data from the MASS package (this doesn’t need installing). The help file (see ?MASS::leuk) explains:

Survival times are given for 33 patients who died from acute myelogenous leukaemia. Also measured was the patient’s white blood cell count at the time of diagnosis. The patients were also factored into 2 groups according to the presence or absence of a morphologic characteristic of white blood cells. Patients termed AG positive were identified by the presence of Auer rods and/or significant granulation of the leukaemic cells in the bone marrow at the time of diagnosis.

head(MASS::leuk)
    wbc      ag time
1  2300 present   65
2   750 present  156
3  4300 present  100
4  2600 present  134
5  6000 present   16
6 10500 present  108

The aim is to fit an exponential AFT model with log of white blood cell count, and ag as explanatory variables.

There is no censoring in this data. We first set up a survival object:

library(survival)
leuk_sv <- Surv(MASS::leuk$time)
Note

For fitting in R, we define \[ \beta_0:=- \log \lambda, \] and then write \[T \sim Exp(\theta(\mathbf{x})),\] where \[\theta(\mathbf{x}) = \exp \left( -(\beta_0 + \beta_1 x_1 + \ldots + \beta_p x_p) \right).\]

In the example, we have \(p=2\) given the two explanatory variables, and we define

  • \(x_1\): the log of the white blood cell count (log(wbc))
  • \(x_2\): an indicator variable, taking the value 1 if the ag variable is present and 0 otherwise.

We have three parameters to estimate: an ‘intercept’ \(\beta_0\) and the coefficients of \(x_1\) and \(x_2\), denoted by \(\beta_1\) and \(\beta_2\) respectively.

Note

A positive coefficient \(\beta_i\) means that increasing \(x_i\) is associated with longer survival times.

We fit this model in R as follows:

leuk_regexp <- survreg(leuk_sv ~ log(wbc) + ag,
                       dist = "exponential",
                       data = MASS::leuk)
summary(leuk_regexp)

5.5.3 Hypothesis testing

Suppose we wish to investigate whether particular explanatory variables have an effect on the survival distribution. We can consider testing hypotheses of the form \[ H_0:\beta_i = 0 \]

We can perform partial z-tests: we compute \[ Z_{obs}:= \frac{\hat{\beta}_i}{\mbox{estimated standard error }(\hat{\beta}_i)}, \] with \(Z_{obs}\sim N(0, 1)\) if \(H_0\) is true.

The tests are conditional on all the remaining covariates being included in the model and hence are termed partial z-tests.

TipExample

To the test the significance of the ag variable (the “presence of Auer rods and/or significant granulation of the leukaemic cells in the bone marrow at the time of diagnosis”), we have

\[ H_0: \beta_2 = 0, \quad Z_{obs} = \frac{1.018}{0.364}=2.80, \] and a \(p\)-value of \[ 2\times P(Z\ge 2.80),\quad \mbox{ where } Z\sim N(0,1), \] which is 0.0051: strong evidence against \(H_0\): there is strong evidence that an ag test result of present is associated with longer survival times.

5.5.4 Predicting survival times

Suppose we want to predict survival for patient with a particular set of covariates \(\mathbf{x}^*\). For this patient’s survival time \(T\), we would have \[ T\sim Exp(\theta(\mathbf{x}^*)), \] and we can estimate \(\theta(\mathbf{x}^*)\) by \[ \hat{\theta}(\mathbf{x}^*):=\exp \left( -(\hat{\beta}_0 + \hat{\beta}_1 x^*_1 + \ldots + \hat{\beta}_p x^*_p) \right). \] We can then estimate the mean survival \(E(T)\) by \[ \hat{E}(T):=\frac{1}{\hat{\theta}(\mathbf{x}^*)} \] and any percentile from the distribution of \(T\) by inverting the estimated cumulative distribution function. For example, to estimate the 95th percentile \(t_{(0.95)}\) of the distribution of \(T\), we note \[ 0.05 = \exp(-t_{(0.95)}\theta(\mathbf{x}^*)), \] and so an estimate would be \[ \hat{t}_{(0.95)} = \frac{-\log 0.05}{\hat{\theta}(\mathbf{x}^*)} \]

TipExample

We estimate the mean survival time for a patient with a wbc of 10000 and who has an ag test result of present. We have \[\mathbf{x^*} = (x_1^*,\,x_2^*)^T=(\log 10000,\, 1)^T\] and \[\hat{\lambda}(\mathbf{x^*}) = \exp(- 5.815 + 0.304\log(10000) - 1.018)= 0.01772,\] hence the estimated mean survival is \(1/0.01772\simeq 56\) days.

Note

Approximate standard errors for these estimates can obtained using the general approximation \[ var(g(\hat{\phi}))\simeq \left.\{g'(\phi)\}^2\right|_{\phi=\hat{\phi}}var(\hat{\phi}), \] with \[ \hat{\phi} = \hat{\beta}_0 + \hat{\beta}_1 x^*_1 + \ldots + \hat{\beta}_p x^*_p, \] and, for example for \(E(T)\), \[ g(\hat{\phi}) = \exp(\hat{\phi}). \] Standard errors can be obtained from R: you will not be required to compute these by hand!

5.5.5 Obtaining predictions using R

Predictions can be obtained using R, similar to how predictions are in obtained in R for linear models.

We first set up a new data frame with the desired explanatory variables for prediction. Column names must match those used for model fitting. Consider again estimating the mean survival time for a patient with a wbc of 10000 and who has an ag test result of present.

xstar <- data.frame(wbc = 10000, ag = "present")

We then use the predict() function:

5.6 The Weibull AFT Model

We briefly discuss the Weibull AFT model. Recall our Weibull parameterisation in Section 1.5.2: \[\begin{align} f(t) &= \lambda \gamma (\lambda t) ^ {\gamma-1} \exp\left[ -(\lambda t) ^ \gamma \right] \\ S(t) &= \exp\left[ -(\lambda t) ^ \gamma \right] \\ h(t) &= \lambda \gamma (\lambda t) ^ {\gamma-1} \end{align}\]

If we model \(T\) as an AFT Weibull model with baseline \(T_0 \sim Weibull(\lambda_0, \gamma)\) then

\[ S(t; \mathbf{x}) = \exp\left(- \left(\lambda_0 t \exp(-\boldsymbol{\beta}' \mathbf{x})\right)^\gamma\right). \] We again define \(\beta_0 = -\log \lambda\), so that \[ S(t; \mathbf{x})= \exp\left(- \left(t \exp(-(\beta_0 + \beta_1 x_1 + \ldots + \beta_p x_p)) \right)^\gamma\right) \] This is equivalent to \(T \sim Weibull(\lambda(\mathbf{x}), \gamma)\) where \[\lambda(\mathbf{x}) = \exp\left(-(\beta_0 + \beta_1 x_1 + \ldots + \beta_p x_p) \right)\] Note that \(\gamma\) does not depend on \(\mathbf{x}\).

We fit this model in R as follows (though weibull is the default argument for dist, so we could leave out dist = "weibull")

leuk_regweib <- survreg(leuk_sv ~ log(wbc) + ag,
                       dist = "weibull",
                       data = MASS::leuk)
summary(leuk_regweib)

The reciprocal of the \(\gamma\) parameter is referred to as Scale (and Log(scale) for \(\log 1/\gamma\)). We see that \(\log 1/\hat{\gamma} = 0.0399\), and the partial \(z\)-test shows \(\log 1/\gamma\) is not significantly different from 0. As \(\gamma=1\) corresponds to the exponential distribution, we would conclude there is no evidence in favour of the Weibull distribution over the simpler exponential distribution.

5.6.1 Mean survival time and predictions

For the Weibull distribution written as \(T \sim Weibull(\lambda, \gamma)\) and parameterised as in Section 1.5.2, the mean survival time is \[ E(T) = \frac{1}{\lambda}\Gamma\left(1+\frac{1}{\gamma}\right), \] where \(\Gamma\) is the gamma function. Note that for integer \(n\), we have \(\Gamma(n) = (n-1)!\), and so for \(\gamma=1\) which gives the exponential distribution, we have \(E(T)=1/\lambda\).

For a patient with covariates \(\mathbf{x}\), in the Weibull AFT model we can estimate their mean survival time by \[ \hat{E}(T) = \frac{1}{\hat{\lambda}(\mathbf{x})}\Gamma\left(1+\frac{1}{\hat{\gamma}}\right) \]

TipExample

We again estimate the mean survival time for a patient with a wbc of 10000 and who has an ag test result of present. We have \[\mathbf{x^*} = (x_1^*,\,x_2^*)^T=(\log 10000,\, 1)^T\] and \[\hat{\lambda}(\mathbf{x^*}) = \exp(-5.8524 +0.3103\log(10000) - 1.0206)= 0.01804\] We also have \(1/\hat{\gamma}=1.04\). We compute the estimated mean survival as

1/0.01804 * gamma(2.04)
[1] 56.40664

which is very similar to the prediction from the exponential model.

Predictions for the Weibull AFT model can also be obtained using the predict() function. However, the default prediction is not the mean survival: it is just the estimate of \(1/\hat{\lambda}(\mathbf{x^*})\)

xstar <- data.frame(wbc = 10000, ag = "present")
predict(leuk_regweib, xstar)
       1 
55.40983 
# compare with estimate of 1/lambda(x)
# (will be rounding error as I've used rounded values of parameter estimates)
1/exp(-5.8524 +0.3103*log(10000) - 1.0206) 
[1] 55.42503

This is not very easy to interpret in terms of the distribution of \(T\), so if using the predict command, it is better to obtain an estimate of the median survival instead:

predict(leuk_regweib, xstar, type = "quantile", p = 0.5)
       1 
37.83864
